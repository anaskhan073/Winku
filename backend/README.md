# Project Setup & Installation

This guide covers the installation of required Node.js packages for the project.

## Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher recommended)
- [npm](https://www.npmjs.com/) (comes with Node.js)
- MongoDB (local or cloud instance via [MongoDB Atlas](https://www.mongodb.com/cloud/atlas))

---

## Installation

Run the following command to install all required dependencies:

```bash
npm i cors express mongoose bcrypt dotenv cookie-parser twilio nodemailer node-cron
